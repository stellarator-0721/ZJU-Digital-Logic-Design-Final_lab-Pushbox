#浙江大学数字逻辑设计课程大作业

ZJU 浙江大学 数逻 数字逻辑设计 Digital Logic Design Pushbox HellTaker

本仓库为本人在浙江大学数字逻辑设计课程中完成的课程大作业代码，使用Verilog语言和学校提供的FPGA板。本实验模仿游戏HellTaker，素材取自游戏截图，实现了其在FPGA上的部署。

本实验代码并不完全完善，在实验过程中我们对地图的实现方法是，将地图寄存器的不同位置赋予不同变量，代表不同方块，然后将对应方块显示在VGA上。但是，在变量固定时，会存在某几个特殊的位置，其上的方块被VGA显示为黑色，具体原理不明，改变变量赋值可能导致VGA重新出现，也可能不会，给我们造成了很大困扰。由于在某一张地图中，最后的终点一直无法显示，我们不得不使用另一个图案重新表示终点。（这就是为什么你能在上面看到原神图标，绝对不是因为我的队友玩原神）
